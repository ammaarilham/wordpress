Daddy Plus is a useful plugin for WordPress theme by Themes Daddy.

== Description ==

This Daddy Plus plugin is created for Themes Daddy. 
The plugin add custom frontpage sections for Themes Daddy and it will allow you full customizer settings for the theme front page sections. 
The plugin options are user-friendly and you can use this plugin for Themes Daddy WordPress theme.
